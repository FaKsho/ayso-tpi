## Resumen del trabajo
 
Se configuró una máquina virtual (Ubuntu Server 22.04) en VirtualBox, en modo de red *bridge*, y se utilizó **ZeroTier** para habilitar el acceso remoto entre dos PCs. Sobre la VM se instaló **nginx**, sirviendo una página estática personalizada, y se ejecutó un script en **Python** en forma de prueba, validando el acceso y la administración remota mediante **SSH**.
 
Los detalles de la realización del trabajo se pueden ver en el pdf del informe: [`TPI-AYSO.pdf`](./TPI-AYSO.pdf).
